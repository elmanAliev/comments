# Тестовое задание

Ссылка на страницу: http://elmanaliev.github.io/comments/

## В проекте применяется
* React
* HTML
* CSS
* JavaScript
* Styled Components

## Запуск проекта

1. Клонировать репозиторий

    `git@github.com:elmanAliev/comments.git`

2. Установить зависимости

    `cd comments`
    
    `npm install`

3. Запуск проекта

    `npm start`
